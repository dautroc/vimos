# VimOS

VimOS is a Swift executable package designed to bring Vim-like modal editing capabilities to your operating system environment.

## Overview

This project implements a Vim engine in Swift, handling various states (Normal, Insert, Visual) and motions to mimic the efficiency of Vim anywhere.

## Usage

As a Swift package, you can run the executable using:

```bash
swift run
```

## Structure

- **VimOS**: The main executable target.

## Requirements

- Swift 6.0+
